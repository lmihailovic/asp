Уредити низ бројева тако да најмањи од свих елемената иде на прво место,
најмањи од преосталих на последње, следећи најмањи по величини на друго место,
следећи на претпоследње итд.
(https://petlja.org/biblioteka/r/Zbirka2/cik_cak_selekcija)

Пример:
```
Улаз                            Излаз
[-3, 7, 1, 8, -5, 2, 9]         [-5, 1, 7, 9, 8, 2, -3]
```
