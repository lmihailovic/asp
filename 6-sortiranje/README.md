# Сортирање

## Задаци

1. [Дат је низ природних бројев. Одредити два броја чија је апсолутна разлика
најмања. Уколико постоји више решења исписати пар који је већи.](zad1/README.md)
2. [Дато је N пакета чоколаде и за сваки од њих је познато колико чоколадица
садржи. Сваки од K ученика узима тачно један пакет, при чему је циљ да сви
ученици имају што приближнији број чоколадица. Колика је најмања могућа разлика
између оног ученика који узме пакет са најмање и оног који узме пакет са
највише чоколадица.](zad2/README.md)
3. [Уредити низ бројева тако да најмањи од свих елемената иде на прво место,
најмањи од преосталих на последње, следећи најмањи по величини на друго место,
следећи на претпоследње итд.](zad3/README.md)
4. [Дат је низ целих бројева у коме је једна половина елемената парна, а друга
непарна. Преуредити низ тако да парни елементи буду на парним индексима, а
непарни на непарним.](zad4/README.md)
5. [Дата су два низа целих бројева. Написати функцију која враћа нови низ који
садржи елементе који се налазе у оба низа (без дупликата).](zad5/README.md)
6. [Дат је низ целих бројева и цео број K. Одредити најмањи могући број
различитих бројева у низу након брисања K елемената.](zad6/README.md)
7. Дат је низ позитивних целих бројева А. За сваку тројку 0 ≤ i < j < k ≤ n
одредити најмању вредност од три броја Ai, Aj, Ak, а затим одредити збир тако
добијених вредности у свим тројкама.
8. Људи су долазили и одлазили са базена и за сваког од N посетилаца је познато
време доласка и време одласка. Претпоставићемо да је човек на базену у периоду
облика [a,b), тј. да се човек налази на базену у тренутку свог доласка a, а да
се не налази у тренутку свог одласка b. Колико је људи највише било истовремено
на базену?
