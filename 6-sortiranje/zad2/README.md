Дато је N пакета чоколаде и за сваки од њих је познато колико чоколадица садржи.
Сваки од K ученика узима тачно један пакет, при чему је циљ да сви ученици
имају што приближнији број чоколадица. Колика је најмања могућа разлика између
оног ученика који узме пакет са најмање и оног који узме пакет са највише чоколадица.  
(https://petlja.org/biblioteka/r/Zbirka2/pravedna_podela_cokoladica)

Пример:
```
Улаз                            Излаз
K = 4
[3, 8, 1, 17, 4, 7, 12, 9]      5
```
