Дат је низ целих бројева у коме је једна половина елемената парна, а друга
непарна. Преуредити низ тако да парни елементи буду на парним индексима, а
непарни на непарним.
(https://leetcode.com/problems/sort-array-by-parity-ii/)

Пример:
```
Улаз                            Излаз
[2, 4, 3, 6, 8, 7, 5, 1]        [2, 3, 4, 5, 8, 7, 6, 1]
```
