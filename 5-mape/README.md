# Хеш мапе
## Задаци

1. [Написати функцију која одређује да ли низ целих бројева садржи дупликате.](zad1/README.md)
2. [Написати функцију која за дати низ целих бројева и цео број Х одређује
позиције два елемента чији је збир једнак Х.](zad2/README.md)
3. [Дат је низ целих бројева. Одредити највеће растојање између два иста
елемента у том низу.](zad3/README.md)
4. [Написати функцију која за дати низ целих бројева одређује колико има
поднизова чија је сума једнака датом броју K.](zad4/README.md)
5. [Дата су два стринга S и Т, испитати да ли су они анаграми. Два стринга су
анаграми ако се један стринг може добити пермутацијом слова другог.](zad5/README.md)
6. [Дати су стринг S и стринг Т. Написати функцију која проналази најмањи
подстринг у речи S такав да садржи све карактере из Т.](zad6/README.md)
7. [Написати функцију која проверава да ли је дати судоку валидан.](zad7/README.md)
