Написати функцију која за дату листу и број Х преуређује елементе тако да се
сви елементи који су мањи од Х налазе пре свих оних коју су већи или једнаки од Х.  
(https://leetcode.com/problems/partition-list/)

Пример:
```
Улаз                                    Излаз
Х = 3
2 -> 1 -> 4 -> 3 -> 3 -> 2 -> 5         2 -> 1 -> 2 -> 4 -> 3 -> 3 -> 5
```
